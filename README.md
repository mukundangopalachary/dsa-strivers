# Strivers A2Z DSA Sheet

## To run project

To run this project, enter the following command in the root directory `dsa-stivers/` of this project:

```bash
mvn exec:java -Dexec.mainClass="com.strivers.App"
```

## Installation Guide

If you haven't installed `maven`:

```bash
sudo apt install maven
```

If your distro isn't Debian, check: [https://maven.apache.org/install.html](https://maven.apache.org/install.html)
